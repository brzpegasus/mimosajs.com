Installation it's a piece of cake

1) node.js

Mimosa is written in node.js and you'll need to have it installed. You'll want to download and install the latest and greatest version. Anything over 8.0 will do. The node.js install comes with the Node Package Manager (npm).

2) mimosa

You use npm to install node packages.

$ npm install -g mimosa

The -g flag installs Mimosa globally so it can be run from the command line anywhere.

3) (You're done)

Quick Start you'll be coding soon

1) Create a new application

The easiest way to get started with Mimosa is to use it to create a new application skeleton. By default, Mimosa will create a basic Express app configured to match your desired meta-langauges.

First navigate to a directory within which you want to place your application. Create the default app:

$ mimosa new nameOfApplicationHere

Follow the prompts and choose the meta-languages you'd like to use. If you aren't sure which ones to choose, that's perfect. Mimosa is a great tool to learn these languages.

2) Start the server

Change into the directory that was created and execute:

$ mimosa watch --server

In your browser navigate to http://localhost:3000 to see the sample app.

Mimosa will watch the assets directory of your new project and compile the changes you make over to the public directory. The web app serves up the assets in the public directory.

3) Start Coding
